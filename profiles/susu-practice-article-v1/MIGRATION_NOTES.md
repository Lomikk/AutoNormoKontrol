# Заготовка профиля статьи

Идентификатор: susu-practice-article-v1

Перенесены восстановленные файлы:

- starter/content/article.md
- starter/metadata.yaml
- starter/bibliography.bib
- templates/practice-article.tex
- styles/practice-article.sty

Профиль пока не зарегистрирован в profiles/catalog.json.

Перед регистрацией необходимо:

1. создать profile.yaml;
2. нормализовать metadata.yaml;
3. добавить минимальные требования статьи;
4. подготовить Lua-фильтр и postflight;
5. проверить совместимость шаблона с текущим build.ps1;
6. добавить интеграционный тест.
