# Контракт агента реферата

Эта папка — одна учебная работа. Редактируй только пользовательские
файлы: `metadata.yaml`, `bibliography.bib`, `content/*.md`, `assets/` и по
прямому запросу — журналы `compliance/`.

Перед работой прочитай `GEMINI.md` и
`guide/profile-system-prompt.md`. Не читай и не меняй движок, профиль
и нормативный корпус. После законченной правки выполни:

```powershell
.\AutoNormoKontrol.cmd draft --quiet
```

Не выдумывай источники, персональные данные, подписи и внешние
решения. Strict не является обязательным для черновой работы.
